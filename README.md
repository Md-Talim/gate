# Gate - URL Shortener

A modern URL shortening service that transforms long URLs into short, shareable links.

## About

Gate is a full-stack URL shortener built to demonstrate clean architecture, RESTful API design, and modern web development practices. It provides a simple, intuitive interface for creating and managing shortened URLs.

## Features

- Shorten long URLs into compact, shareable links
- Track click analytics for each shortened URL
- Fast redirect performance
- Clean, responsive user interface
- Secure and reliable

## Tech Stack

**Backend:**

- Java 17+
- Spring Boot
- Maven
- PostgreSQL (or H2 for development)

**Frontend:**

- React
- Modern JavaScript/ES6+
